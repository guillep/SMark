# SMark
Write Benchmarks like Tests

## Installation
### Pharo 7+
```smalltalk
Metacello new
    baseline: 'SMark';
    repository: 'github://smarr/SMark';
    load.
```
### Squeak 5.2+
```smalltalk
Installer ensureRecentMetacello.
Metacello new
    baseline: 'SMark';
    repository: 'github://smarr/SMark';
    load.
```
